# Spec 07 — Acceptance Criteria

## Functional
- PGN successfully processed end-to-end
- features generated for all valid moves
- ML predictions available
- insights generated and validated

## Validation
- ≥90% insights must be VERIFIED or WARNING
- 0 insights without evidence
- deterministic outputs for same input

## Performance
- configurable max processing time per game

## Quality
- no contradiction between:
  - engine evaluation
  - ML classification
  - explanation
